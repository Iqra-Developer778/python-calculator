# 🧮 Python Calculator

🐍 Built with Python  🚀

## 📋 About
A simple command-line calculator built with Python.
Handles basic arithmetic operations with input validation and error handling.

## ⚡ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ✅ Division by zero error handling
- ✅ Invalid input validation (only numbers allowed)

## 🚀 How to Run
```bash
python calculator.py
```

## 🛠️ Built With
- Python 3

## 📌 Operations Menu
| Choice | Operation |
|--------|-----------|
| 1      | Add (+)   |
| 2      | Subtract (-) |
| 3      | Multiply (*) |
| 4      | Divide (/) |
| 5      | Exit      |
